DELIVERY MODEL

作ったアプリを、
チーム全員で使える環境へ

Claude Codeで構築したアプリは、担当者1人のPC上だけでなく、AWSを活用して 社内全員がブラウザからアクセスできる本番環境として展開できます。 用途・規模・コストに合わせて3つの納品モデルからお選びいただけます。

PATTERN 01
💻
ローカルPC 納品
担当者のPCにインストールして利用。サーバー費用ゼロ。
  • インストール・設定レクチャー込み
  • 追加サーバー費用ゼロ
  • API費用のみ実費
  • 複数人での共有は別途対応が必要
現行プランの標準形式
軽量・低コストに最適
PATTERN 02
⚡
AWS Lambda
サーバーレス展開
使った分だけ課金。AIへの問い合わせ・フォーム処理など軽量用途に。
  • 月100万リクエストまで無料枠あり
  • サーバー管理が不要
  • 全員がブラウザでアクセス可能
  • データ保存が少ない用途に最適
  • 大量データ処理・常時接続には不向き
小〜中規模チームに推奨
フル機能・全社展開に
PATTERN 03
☁️
AWS EC2
フルサーバー展開
自社AWSに専用サーバーを構築。大規模・データ蓄積・複雑な処理に対応。
  • データをDBに蓄積・管理できる
  • 複雑な処理・大量データに対応
  • 権限管理・ログ管理が充実
  • 全員がブラウザでアクセス可能
  • カスタマイズの自由度が最も高い
中〜大規模・本格運用に

🔒 データセキュリティについて
🏢
データは自社AWSに保存
業務データはすべて貴社が契約するAWSアカウント内に保存。弊社や第三者がアクセスすることはありません。
🔑
アクセス管理・権限設定
ログインID・パスワードで従業員を管理。管理者・一般ユーザーなど権限ごとに表示できる情報を制御できます。
🌐
SSL暗号化・HTTPS通信
すべての通信はSSL証明書（HTTPS）で暗号化。社外からのアクセスも安全に利用できます。
